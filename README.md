# SimpleV
My own sorting algorithm visualizer

It was made in a week and it doesn't have GUI, but it's powerful enough to create showcase videos and even benchmark algorithms

## Compile
To compile the source code run `compile.bat` or just `compile` in cmd

## Run
To run you need to use `run.bat` and pass a suite as an argument, for example: `run suite.txt`

look at suite1-3.txt for examples

### Important notice
The algorithms are compiled at runtime, so you don't need to recompile the entire project every time you change something in an algorithm
